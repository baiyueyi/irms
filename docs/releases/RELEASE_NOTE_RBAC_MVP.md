# IRMS RBAC MVP 发布说明（最小版）

## 本轮完成了什么

- 交付 RBAC MVP 主链路：用户/用户组/资源/资源组/授权/权限计算/审计日志
- 完成首次登录强制改密（超管与普通用户）
- 完成 SuperAdmin CLI 初始化（含重复初始化拦截）
- 完成普通用户“我的可访问资源”页面与 `GET /api/permissions/resources`
- 完成 PowerShell 端到端联调并落盘测试报告
- 后端工程目录归位到 `backend/`（命令统一在该目录执行）
- 修复改密后仍被 `password change required` 拦截的问题（首次改密状态按服务端实时用户状态判定）

## 还没做什么

- 未实现多租户、审批流、字段级权限、外部系统集成
- 未实现复杂报表与高级检索

## 已知限制

- `audit_logs.ip` 在部分本地代理链路下可能记录为非标准值，后续可增强
- 前端为 MVP 交互形态，仅覆盖验收所需编辑/删除/关键筛选

## 下一阶段范围

- 稳定性收敛：错误码一致性、审计内容细化、权限边界回归
- 体验收敛：管理页表单校验与操作反馈优化
- 发布准备：内网部署脚本与回归用例固化
