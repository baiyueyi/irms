# Rule 03｜技术栈固定

- 前端固定使用：
  - `Vite`
  - `Vue 3`
  - `Pinia`
  - `Element Plus`
- 后端固定使用：
  - `Golang`
  - `gin`
  - `gorm`
  - `gorm/gen`
  - `swagger`
- 数据库固定使用：
  - `MySQL`
- 未经明确批准，不引入新框架，不替换现有技术栈。
- 页面组件优先使用 Element Plus 现有能力，不额外引入重型 UI 框架。
- 后端实现优先保持简单、可读、可联调，不做平台级抽象。
- 新链路不再引入 `chi` 或新的并行 HTTP 主框架。
